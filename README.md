Explorer Utilitys
================

Some usefull Windows Explorer extensions and context menue entrys.

## Supported OS (x86, x64):
- Windows Vista SP2+
- Windows 7 SP1+
- Windows 8
- Server 2008 + R2
- Server 2012

## How to use:
Compile with VS2012 
Install: regsvr32 ExplorerUtilitys_x86.dll
Uninstall: regsvr32 -u ExplorerUtilitys_x86.dll 

- - - - - - - - - - - - - - - - - - - -
